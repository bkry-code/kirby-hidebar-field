# Kirby Hidebar Field

**⚠️ Work in progress...**

Kirby field which hides the sidebar and extends the content area full width.

Best used in combo with [kirby-index-field](https://github.com/jongacnik/kirby-index-field)

## Usage

```yaml
hidebar:
  type: hidebar
```
